Learning From DataвҖ”вҖ”derive something from Softmax

Posted on 2018-12-05 Edited on 2025-09-18 In ж•°жҚ®еӯҰд№ иҜҫзЁӢ

иҝҷе‘Ёзҡ„ж•°жҚ®еӯҰд№ иҜҫжӣҙдёҚзҹҘйҒ“иҜҘиө·д»Җд№Ҳйўҳзӣ®дәҶгҖӮдё»иҰҒжҳҜеҠ дёҠдёҖдәӣеҒҮи®ҫпјҢд»ҺSoftmaxеҮҪж•°ејҖе§ӢжҺЁеҜјпјҢжңҖеҗҺеҫ—еҲ°дёҖдёӘйқһеёёз®ҖеҚ•зҡ„еҪўејҸпјҢд»ҺиҖҢеӨ§еӨ§з®ҖеҢ–дәҶз®—жі•гҖӮиҝҷж¬Ўзҡ„derivationе’ҢдёҠзҜҮи®Іеҫ—дёңиҘҝиҝҳжҳҜжңүдёҖдәӣзӣёе…ізҡ„гҖӮ

Review

йҰ–е…ҲеӣһйЎҫдёҖдёӢдёҠзҜҮеҚҡе®ўд»Ӣз»Қзҡ„еҶ…е®№пјҢд»ҺHGR maximal correlationејҖе§ӢжҺЁеҜјгҖӮдҫқз„¶жҳҜзҰ»ж•ЈеҸҳйҮҸ$X$дёҺ$Y$гҖӮдёҚиҝҮзЁҚеҫ®еҒҡзӮ№жӢ“еұ•пјҢжҲ‘д»¬еңЁжҸҗеҸ–пҪҳпјҢпҪҷзҡ„дҝЎжҒҜж—¶пјҢжҠҠд»–д»¬жҳ е°„еҲ°дёҖдёӘпҪӢз»ҙеәҰзҡ„еҗ‘йҮҸпјҢд№ҹе°ұжҳҜпјҡ

иҝҷж—¶еҖҷпјҢе’Ңд№ӢеүҚдёҖж ·пјҢеҒҡдёҖдәӣжҺЁеҜјеҗ§гҖӮиҝҷж—¶еҖҷзҡ„зӣёе…ізі»ж•°еҸҳжҲҗдәҶзӣёе…ізҹ©йҳөпјҡ

жҲ‘д»¬зҡ„зәҰжқҹеҸҳжҲҗпјҡ

й—®йўҳжҸҸиҝ°еҸҳдёәпјҡ

е…¶дёӯпјҡ

иҖҢиҝҷж—¶еҖҷзҡ„$\Phi$дёҺ$\Psi$е®һйҷ…дёҠжҳҜз”ұпјўзҡ„з¬¬$2пјҢвҖҰ,k+1$еҸіе·Ұзү№еҫҒеҗ‘йҮҸз»„жҲҗ:

$f(x)=\frac{\phi(x)}{\sqrt{p_X(x)} } ,g(y) = \frac{\psi(y)}{\sqrt{p_Y(y)} }$.

HGR & Softmax

еҒҮи®ҫ$X,Y$жҳҜзҰ»ж•Јзҡ„пјҢе№¶дё”еҮ д№ҺзӢ¬з«Ӣ(weakly dependent),д№ҹе°ұжҳҜ$p_ {XY}(xy) - p_ {X}(x)p_ {Y}(y)$йқһеёёе°ҸгҖӮ

иҝҳи®°еҫ—softmax function:

еңЁиҝҷйҮҢпјҢжҲ‘д»¬жҠҠ$X,Y$еҶҚж¬ЎиҝӣиЎҢдҝЎжҒҜжҸҗеҸ–пјҢеҲҶеёғдёә$f(X),g(Y)$. з”ұдәҺ$W_y$дёҺYеҖјзӣёе…іпјҢжҲ‘д»¬еҸҜд»Ҙе°Ҷ$W_y$зңӢдҪңжҳҜg(y)гҖӮеӣ жӯӨеҶҷжҲҗжӣҙйҖҡз”Ёзҡ„еҪўејҸпјҡ

зҺ°еңЁжҲ‘д»¬е®ҡд№үпјҡ$d(y) \triangleq b(y)-\log p_Y(y)$пјҢеҲҷпјҡ

еҸҜд»ҘзңӢеҲ°пјҢеҰӮжһң$f = g = d = 0$пјҢ$Q_ {Y|X}(y|x) = p_Y(y)$.

з”ұдәҺжҲ‘д»¬зҡ„еҒҮи®ҫеҸҜд»ҘзҹҘйҒ“,$p_Y(y) \approx Q_ {Y|X}(y|x)$пјҢеҲҷ$f^T(x)g(y)+ d(y) \approx 0$,ж №жҚ®жі°еӢ’еұ•ејҖ:

иҖҢ:

иҖҢз”ұжі°еӢ’еұ•ејҖ$\frac{1}{1+x} \approx 1-x$еҫ—еҲ°пјҡ

з»“еҗҲдёҠйқўзҡ„(1),(2)пјҢжҲ‘д»¬еҫ—еҲ°пјҡ

зҺ°еңЁжҲ‘д»¬д»Ө$\tilde{g}(y) = g(y) - \mathbb{E}_Y[g(Y)]пјҢs.t. \mathbb{E}_Y[\tilde {g}(Y)] = 0$.еҫ—еҲ°пјҡ

зҺ°еңЁжҲ‘д»¬еҲ©з”ЁиҝҷдёӘејҸеӯҗжһ„е»ә$empirical risk$пјҢе®һйҷ…дёҠд№ҹе°ұжҳҜ$-\frac{1}{n} \sum_ {i=1}^n \log Q_ {Y|X}(y_i|x_i)$.жңҖе°ҸеҢ–з»ҸйӘҢйЈҺйҷ©(empirical risk)е®һйҷ…дёҠд№ҹе°ұжҳҜжңҖеӨ§еҢ–$\mathbb{E}_ {p_ {XY} } [ Q_ {Y|X}(y|x)]$пјҢд№ҹжҳҜжһҒеӨ§дјјз„¶дј°и®ЎгҖӮ

дёҠиҝ°иҝҮзЁӢз”ЁеҲ°дәҶжі°еӢ’еұ•ејҖпјҡ$\log(1+x) \approx x - \frac{x^2}{2}$.

$\mathbb{E}[\log Q_ {Y|X}(Y|X)] = \mathbb{E}[\log p_Y(Y)] + \mathbb{E}[f^T(X)\tilde{g}(Y) ] + \mathbb{E}[\tilde{d}(Y) ] - \mathbb{E}[\frac{1}{2} [(f^T(X)\tilde{g}(Y))^2 + \tilde{d^2}(Y) + 2 f^T(X)\tilde{g}(Y)\tilde{d}(Y)]] $

зҺ°еңЁпјҢжҲ‘д»¬жқҘиҜҙжҳҺдёҖдәӣеҝ…иҰҒзҡ„дёңиҘҝпјҡз”ұеҒҮи®ҫеҫ—еҲ°$p_ {XY}(x,y) - p_X(x)p_Y(y) = \epsilon \cdot \square = o(\epsilon)$пјҢ$o(\epsilon)$иЎЁзӨә$\epsilon$зҡ„ж— з©·е°ҸйҮҸпјҲиҝҷд№ҲиҜҙе…¶е®һдёҚеҮҶзЎ®пјҢеӣ дёәжҲ‘д»¬жңҖеҗҺиҰҒжңҖеӨ§еҢ–иҝҷз§Қж— з©·е°ҸйҮҸпјҢжҳҫз„¶дёҚеҗҲзҗҶпјҢеҸҜд»ҘеҪ“дҪңдёәиЎЎйҮҸжңүеӨҡе°Ҹзҡ„йҮҸзә§пјүгҖӮ

еӣ дёә
еҗҢзҗҶеҸҜд»Ҙеҫ—еҲ°$f^T(x)\tilde g (y) = o(\epsilon),\tilde d (y) = o(\epsilon)$пјҢжҲ‘д»¬еҒҮи®ҫеҜ№жүҖжңүзҡ„$f,\tilde g,\tilde d$йғҪиҝӣиЎҢдәҶnormalizeпјҢд№ҹе°ұжҳҜ$\mathbb{E}f = \mathbb{E}\tilde{g} = \mathbb{E}\tilde{d} = 0$,еҲҷпјҡ

иҖҢжҲ‘д»¬зҹҘйҒ“$o(\epsilon^3)$еңЁ$o(\epsilon^2)$д№ӢеүҚжҳҜеҸҜд»Ҙиў«еҝҪз•Ҙзҡ„гҖӮеӣ жӯӨжңҖз»Ҳпјҡ

дёҠејҸдёӯпјҢз¬¬дёҖйЎ№дёәеёёж•°пјҢжңҖеҗҺдёҖйЎ№дёәйқһиҙҹеҖјпјҢдё”дёҺеүҚйқўеҮ йЎ№жІЎжңүзәҰжқҹе…ізі»пјҢеӣ жӯӨдёәдәҶжңҖеӨ§еҢ–дёҠејҸеҸӘйңҖз®ҖеҚ•д»Ө$\frac{1}{2} \mathbb{E}[\tilde{d^2} (y)]=0$пјҢеӣ жӯӨжңҖз»ҲжҲ‘д»¬иҰҒеҒҡзҡ„жҳҜпјҡ

еҰӮжһңжҲ‘д»¬е°Ҷ$ \mathbb{E}[f^T(x)\tilde g(y)] $еҜ№$f(x)$жұӮеҜјпјҢеҸҜд»Ҙеҫ—еҲ°пјҡ

е…¶дёӯ$\land ^{-1}_ {\tilde g (Y)}=(\mathbb{E}_ {p_Y}[\tilde g(Y){\tilde g}^T(Y)])^{-1}$пјҢд№ҹе°ұжҳҜжҲ‘д»¬еҫ—еҲ°дәҶжңҖдҪізҡ„$f,\tilde g$.

еҗҢзҗҶжҲ‘д»¬д№ҹеҫ—еҲ°пјҡ

д№ҹе°ұжҳҜеҰӮжһңжҲ‘д»¬еҗ‘softmaxеҮҪж•°дёӯе–Ӯе…Ҙ$f(x)$пјҲеҪўејҸеӣәе®ҡпјү,йӮЈд№Ҳsoftmaxе°ҪйҮҸеңЁеӯҰзҡ„дёңиҘҝпјҢд№ҹе°ұжҳҜ$W$е®һйҷ…дёҠжҳҜ$g^*$пјҢеҪ“з„¶дёҚдёҖе®ҡиғҪжҲҗеҠҹеӯҰеҲ°иҝҷж ·зҡ„еҪўејҸгҖӮ

еҗҢж ·зҡ„пјҢеҲ©з”ЁзҘһз»ҸзҪ‘з»ңиҝӣиЎҢsoftmaxеҸҜд»ҘзңӢдҪңжҳҜеңЁеҜ»жүҫпјёзҡ„зү№еҫҒпјҢе®ғжүҫеҲ°зҡ„жңҖдҪіеҪўејҸеә”иҜҘжҳҜ$f*$.

з®ҖзӣҙжҳҜеӨҙеӨ§гҖӮе®һйҷ…дёҠжҲ‘дёҚиғҪдҝқиҜҒиҝҷзҜҮеҚҡе®ўзҡ„жӯЈзЎ®жҖ§гҖӮ
