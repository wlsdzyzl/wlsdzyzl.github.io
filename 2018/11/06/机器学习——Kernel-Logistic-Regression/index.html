жңәеҷЁеӯҰд№ вҖ”вҖ”Kernel Logistic Regression

Posted on 2018-11-06 Edited on 2025-09-18 In жңәеҷЁеӯҰд№ 

еңЁжҲ‘д»¬зҡ„з”ҹжҙ»дёӯпјҢе…¶е®һеӨ§йғЁеҲҶдҪҝз”Ёзҡ„йғҪжҳҜsoft-margin SVMпјҢеҫҲе°‘дјҡжңүдәәзңҹжӯЈеҺ»дҪҝз”Ёhard-marginпјҢеӣ дёәжҲ‘д»¬ж— жі•йҒҝе…ҚеҷӘеЈ°гҖӮзҺ°еңЁжғіжғіпјҢиғҪеҗҰе°Ҷsoft-margin svmдёҺжҲ‘д»¬д№ӢеүҚзҡ„losgistic regressз»“еҗҲиө·жқҘпјҢдјҡеҫ—еҲ°д»Җд№Ҳж ·зҡ„еӯҰд№ з®—жі•пјҹ

д№ӢеүҚжҲ‘д»¬иҜ•зқҖе°Ҷsoft-margin SVMзҡ„ж•°еӯҰжҸҸиҝ°еҶҷжҲҗдәҶеҸҰеӨ–дёҖз§ҚеҪўејҸпјҡ

min $\frac 1 2 W^TW + C\sum_ {n=1}^N max{1-y_n(W^TX_n+b),0}.$

е®һйҷ…дёҠпјҢиҝҷдёӘеҪўејҸеҰӮжһңдҪ д»”з»Ҷи§ӮеҜҹиҝҷдёӘеҪўејҸпјҢе°ұдјҡеҸ‘зҺ°е®һйҷ…дёҠе®ғе’Ңlosgistic regressionеҠ дёҠL2жӯЈеҲҷеҢ–д№ӢеҗҺзҡ„еҪўејҸйқһеёёзҡ„зӣёдјјпјҡ

algorithm minimize constraint
regularization by constraint $E_ {in}$ $W^TW \leq C$
hard-margin SVM $W^TW$ $E_ {in} = 0$[and more]
L2 regularization $\frac \lambda N W^TW + E_ {in}$
soft-margin SVM $\frac 1 2 W^TW + CN\hat{E_ {in} }$

еҸӘдёҚиҝҮд№ӢеүҚжҲ‘д»¬з”Ё$\lambda$пјҢиҖҢзҺ°еңЁйҮҮз”Ёзҡ„жҳҜдёҖдёӘеёёж•°CгҖӮжҲ‘д»¬зҹҘйҒ“пјҢCи¶ҠеӨ§пјҢд№ҹе°ұжҳҜеҜ№еә”зҡ„$\lambda$и¶Ҡе°ҸгҖӮеңЁиҝҷйҮҢпјҢжҲ‘д»¬е°Ҷ$ max{1-y(W^TX_n+b),0}$зңӢдҪңдёәдёҖз§Қerror measurementгҖӮ

еҰӮжһңз”»еҮәиҝҷеҮ дёӘй”ҷиҜҜзҡ„жӣІзәҝдёҺ0\1й”ҷиҜҜзҡ„еҜ№жҜ”пјҡ
$$

$$

е®һйҷ…дёҠпјҢиҝҷдёӨдёӘеҮҪж•°жҳҜйқһеёёжҺҘиҝ‘зҡ„гҖӮеҪ“$y_n(W^TX_n+b)$еҫҲеӨ§зҡ„ж—¶еҖҷпјҢlogistic regressionзҡ„$E_ {in}(\log_2^{1+e^{-ys} })$е’Ң SVMзҡ„$\hat{E_ {in} }(max{1-y(W^TX_n+b),0})$йғҪжҳҜи¶ӢдәҺ0зҡ„,иҖҢеҪ“$y_n(W^TX_n+b)$йқһеёёе°ҸпјҲиҝңе°ҸдәҺ0пјүзҡ„ж—¶еҖҷпјҢе®ғд»¬зҡ„$E_ {in}$жңүйғҪи¶ӢдәҺ$\vert y_n(W^TX_n+b) \vert$.

жүҖд»ҘжҲ‘д»¬еҸҜд»Ҙи§үеҫ—пјҢе®һйҷ…дёҠSVMе’Ңlogistic regression with L2 regularizationеҮ д№ҺеңЁеҒҡдёҖж ·зҡ„дәӢжғ…гҖӮ

зҺ°еңЁжҲ‘д»¬еёҢжңӣеҸҜд»Ҙе°ҶдәҢиҖ…з»“еҗҲпјҢдҫӢеҰӮжҲ‘д»¬з”ЁSVMзҡ„еҖјжқҘйў„жөӢжҰӮзҺҮпјҢжҲ‘д»¬д№ҹеҸҜд»Ҙеҫ—еҲ°дёҚй”ҷзҡ„з»“жһңгҖӮдҪҶжҳҜиҝҷдёӘе®һйҷ…дёҠжІЎжңүз”ЁеҲ°logisticгҖӮжҲ–иҖ…жҲ‘д»¬з”ЁSVMзҡ„з»“жһңжқҘеҒҡLogitsitc regressionзҡ„еҲқе§ӢеҖјпјҢдҪҶжҳҜж—ўз„¶logistic regressionзҡ„$E_ {in}$жҳҜеҮёеҮҪж•°пјҢеӣ жӯӨе®һйҷ…дёҠеҫ—еҲ°зҡ„жңҖз»Ҳз»“жһңеҢәеҲ«д№ҹдёҚеӨ§пјҢд№ҹе°ұжҳҜе®һйҷ…дёҠе°ұеғҸжІЎжңүз”ЁеҲ°SVMгҖӮ

PlattвҖҷs Model

жңүдёҖз§Қиҝҷж ·зҡ„ж–№жі•пјҢе°Ҷзӣ®ж ҮеҮҪж•°еҶҷдёәпјҡ$g(X) = \theta (A(W_ {SVM}^T \phi(X) + b_ {SVM}) +B)$

еҜ№дёҠйқўзҡ„еҮҪж•°иҝӣиЎҢLogistic RegressionгҖӮ

жүҖд»Ҙиҝҷж—¶еҖҷзҡ„Cost FunctionеҸҳжҲҗпјҡ
$$
min_ {A,B} \frac 1 N \sum_ {n=1}^N \log\left( 1+\exp\left( -y_n(A\cdot(W_ {SVM}^T\phi(X_n)+b_ {SVM})+B)\right)\right)
$$

иҝҷж—¶еҖҷзҡ„cost funtion еҘҪеғҸзңӢдёҠеҺ»йқһеёёеӨҚжқӮпјҢдҪҶжҳҜд»”з»ҶжғідёҖжғізҡ„иҜқпјҢе®һйҷ…дёҠж—ўз„¶жҲ‘д»¬е·Із»ҸжңүдәҶSVMзҡ„з»“жһңпјҢеӣ жӯӨе®һйҷ…дёҠ$W_ {SVM}^T\phi(X_n)+b_ {SVM}$е°ұжҳҜдёҖдёӘеҖјпјҢиҖҢдёҚеҶҚжҳҜдёҖдёӘеҗ‘йҮҸпјҢд№ҹе°ұжҳҜжҲ‘д»¬еҸӘйңҖиҰҒдёӨдёӘж•°зҡ„еҖјпјҡAпјҢBгҖӮе°ұеҸҜд»ҘиһҚеҗҲSVMе’ҢLogistic RegressionгҖӮ

еҪ“з„¶пјҢеҰӮжһңSVMеҒҡзҡ„еҘҪзҡ„иҜқпјҢAзҡ„еҖјеә”иҜҘжҳҜеӨ§дәҺ0зҡ„пјҢBзҡ„еҖјеә”иҜҘжҳҜжҺҘиҝ‘дәҺ0зҡ„пјҢеӣ дёәжҲ‘д»¬еҫ—еҲ°зҡ„жңҖз»Ҳзҡ„еҸӮж•°еҲҶеҲ«дёә$AW_ {SVM}^T$дёҺ$Ab_ {SVM}+AB$пјҢеҲҶеҲ«еҜ№еә”жңҖз»Ҳзҡ„Wе’ҢbпјҢеҰӮжһңSVMеҒҡзҡ„дёҚй”ҷпјҢж„Ҹе‘ізқҖд»–д»¬е’Ң$W_ {SVM},b_ {SVM}$е·®и·қеӨӘеӨ§гҖӮ

дёҠиҝ°дёӯ$\phi(X)$ж„Ҹе‘ізқҖеҲ©з”ЁдәҶзү№еҫҒиҪ¬жҚўпјҢд№ҹе°ұжҳҜдјҡдҪҝз”ЁkernelгҖӮеҸҜд»Ҙеҫ—еҲ°жҜ”иҫғдёҚй”ҷзҡ„logistic regressionеңЁzз©әй—ҙдёҚй”ҷзҡ„и§ЈгҖӮ

дҪҶжҳҜдёҠйқўиҝҷдёӘеҠһжі•пјҢдёҚиғҪдҝқиҜҒиҝҷжҳҜlogistic regressionеңЁZз©әй—ҙпјҲиҪ¬жҚўд№ӢеҗҺзҡ„з©әй—ҙпјүзңҹжӯЈжңҖеҘҪзҡ„и§ЈгҖӮ

Kernal Logistic Regression

жғіиҰҒжүҫеҲ°logistic regressionеңЁZз©әй—ҙзңҹжӯЈжңҖеҘҪзҡ„и§ЈпјҢдёҖдёӘеҠһжі•жҳҜеңЁZз©әй—ҙеҒҡlogistic regressionгҖӮдҪҶжҳҜжҲ‘д»¬зҡ„иҪ¬жҚўе®һйҷ…дёҠжҳҜз”ұkernelжҸҗдҫӣзҡ„пјҢж–№дҫҝи®Ўз®—$Z_n^TZ_m$гҖӮиҖҢlogistic regressionж №жң¬е°ұдёҚжҳҜдәҢж¬Ўи§„еҲ’й—®йўҳпјҢеҸҲеҰӮдҪ•з”ЁеҲ°kernelпјҹ

е…¶е®һпјҢжҲ‘д»¬дёҖзӣҙеңЁжұӮзҡ„дёңиҘҝжҳҜ$W$пјҢ$W$зҡ„з»ҙеәҰжҳҜе’Ң$Z$зҡ„з»ҙеәҰдёҖж ·пјҢйӮЈд№Ҳ$W$жҳҜдёҚжҳҜ$Z$зҡ„зәҝжҖ§з»„еҗҲе‘ўпјҹ

еңЁSVMдёӯпјҢжӯЈжҳҜиҝҷж ·пјҢиҝҳи®°еҫ—$W$жҖҺд№Ҳз®—еҗ—пјҡ
$$
W = \sum_ {n=1}^N \alpha_n y_nX_n
$$

еҗҢж ·зҡ„пјҢеңЁPLAпјҢLogistic Regressionдёӯд№ҹжҳҜиҝҷж ·гҖӮеҒҮеҰӮжҲ‘д»¬жғіжғі$W$зҡ„еҲқе§ӢеҖјдёә0,йӮЈд№ҲжҜҸж¬Ўжӣҙж–°жӯҘйӘӨдёҚе°ұжҳҜдёҖдёӘзі»ж•°д№ҳдёҠдёҖдёӘ$X_i$зҡ„зәҝжҖ§з»„еҗҲеҗ—пјҹ

е®һйҷ…дёҠпјҢеңЁL2зҡ„regularizationдёӯпјҢжңҖеҘҪзҡ„WйғҪжҳҜеҸҜд»Ҙиў«ZзәҝжҖ§з»„еҗҲиЎЁзӨәеҮәжқҘзҡ„гҖӮ

еҰӮдҪ•иҜҒжҳҺиҝҷд»¶дәӢжғ…пјҹ

жҲ‘д»¬е°Ҷoptimal $W$ еҶҷдёә$W^*$.
е…¶дёӯ$W^*$еҸҜд»ҘиЎЁзӨәдёәеһӮзӣҙXз©әй—ҙзҡ„дёҺе№іиЎҢXз©әй—ҙпјҲзәҝжҖ§з»„еҗҲеҚідёәе№іиЎҢпјүзҡ„гҖӮ

$W^* = W_ {\Vert} + W_ {\perp}$

жҲ‘д»¬еҸҜд»ҘиҜҒжҳҺзҡ„жҳҜпјҢеҪ“$W^*$дёӯ$W^{\perp}$дёә0.

еҰӮжһң$W_ {\perp}!=0$,еҲҷ$W^{*T}X = W_ {\Vert}^T X + 0$гҖӮ

д»ҺеҸҰдёҖж–№йқўжқҘиҜҙпјҡ
$W^{ * T}W^* = W_ {\Vert} ^T W_ {\Vert} + W_ {\perp}^TW_ {\perp} >W_ {\Vert} ^T W_ {\Vert} $

иҝҷиҜҙжҳҺ$W_ {\Vert}$жҜ”$W^*$жҳҜжӣҙеҘҪзҡ„йҖүжӢ©пјҢдёҺеҒҮи®ҫзҹӣзӣҫгҖӮ

жүҖд»ҘпјҢ$W^{*} = \sum_ {n=1}^N\beta_nZ_n$.

еӣ жӯӨпјҢжҲ‘д»¬е°ҶWжҚўжҲҗдёҠејҸпјҢйӮЈд№Ҳе°ұдјҡеҮәзҺ°дәҶжҲ‘д»¬жғіиҰҒзҡ„$Z_nZ_m$.

L2 regularizationеҸҳжҲҗдёӢйқўзҡ„ж ·еӯҗпјҡ
$$
min_ {W} \frac \lambda N \sum_ {n=1}^N\sum_ {m=1}^N \beta_n\beta_m Z_n^TZ_m + \frac 1 N \sum_ {n=1}^N \log \left(1 + \exp\left(-y_n \sum_ {m=1}^N \beta_m Z_m Z_n \right) \right)
$$

жҲ‘д»¬еҸҜд»ҘиҪ»жҳ“е°ҶдёҠејҸдёӯзҡ„$Z_n^TZ_m$жҚўжҲҗ$k(x_n,x_m)$.д»ҺиҖҢе®һзҺ°еңЁzз©әй—ҙдёҠlogistic regressionзҡ„жңҖдјҳи§ЈгҖӮ

д»ҺеҸҰдёҖж–№йқўжқҘиҜҙпјҢ$ \sum_ {n}^N\sum_ {m}^N \beta_n\beta_m k(x_n,_m) = \beta ^T K \beta$,еҸҜд»Ҙе°Ҷе®ғзңӢжҲҗдёҖз§Қзү№ж®Ҡзҡ„жӯЈеҲҷеҢ–гҖӮжүҖд»Ҙе®һйҷ…дёҠжҲ‘д»¬еҸҜд»Ҙе°ҶKLRзңӢжҲҗе…ідәҺиҪ¬жҚўеҗҺзҡ„ж•°жҚ®еңЁ$\beta$дёҠзҡ„зәҝжҖ§жЁЎеһӢпјҲеҺҹжқҘжҳҜе…ідәҺWзҡ„зәҝжҖ§жЁЎеһӢпјүгҖӮ

дёҖиҲ¬жқҘиҜҙ$beta_n$йғҪдёҚдёә0,жүҖд»ҘиҝҷдёӘе’ҢSVMдёӯзҡ„$\alpha_n$дёҚдёҖж ·гҖӮ
